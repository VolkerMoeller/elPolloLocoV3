BackgroundObject(imagePath, x)

new BackgroundObject(imagePath, x)

This function causes a new background image to be loaded when an object of this class is created.
Parameters:
Name Type Description
imagePath string – This is the path name under which the corresponding image is found and loaded.
x number – This is the x-coordinate of the position at which the background image is displayed.
