setStopableInterval(fn, ms)

This function saves all individual animation interval IDs in a single array. In addition, the current interval ID is transferred to a variable. This value is passed to the calling objects to uniquely address the object. This establishes the premise that each object only executes one main interval function.
Parameters:
Name Type Description
fn function – This is a function that should be repeated at intervals.
